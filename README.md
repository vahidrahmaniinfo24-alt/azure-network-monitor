Automated Hybrid Network & Monitoring Dashboard
​Dies ist ein Agent-basiertes Netzwerküberwachungssystem, das von mir, Vahid Rahmani, entworfen und implementiert wurde. Das Tool überwacht Systemressourcen (CPU) und die Erreichbarkeit des Netzwerk-Gateways und stellt diese in Echtzeit grafisch dar.

---

​Hauptfunktionen
​Agent-basiertes Monitoring: Erfassung von Live-Systemmetriken.

​Visuelles Dashboard: Anzeige des Netzwerkstatus und der CPU-Auslastung.

​Hybrid Data Flow: Verwendung von JSON-Dateien zum Datenaustausch zwischen Agent und Dashboard.
