💻 VS Code Anleitung: Hintergrundbild oder Animation einfügen

Das <body>-Tag umfasst den gesamten sichtbaren Bereich. Wir können entweder ein echtes Hintergrundbild laden oder eine flüssige Farbanimation erstellen.

Option A: Festes Hintergrundbild (Hintergrund-Grafik)

Mit background-image: url(...) binden wir das Bild ein. Damit es beim Scrollen fixiert bleibt, nutzen wir background-attachment: fixed;. 

body {
    background-image: url('https://images.unsplash.com/...');
    background-size: cover;        /* Füllt den Bildschirm aus */
    background-attachment: fixed;  /* Bild scrollt nicht mit */
    background-repeat: no-repeat;
}

Option B: Animierter Farbverlauf (CSS @keyframes)

body {
    background: linear-gradient(-45deg, #004b87, #252423, #0078d4);
    background-size: 400% 400%;
    animation: gradientAnimation 15s ease infinite;
}

@keyframes gradientAnimation {
    0% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
}

💻 VS Code Anleitung: Header-Bereich (Kopfzeile)

Der Header ist die Überschrift ganz oben auf unserer Webseite. Um diesen zu erstellen, schreiben wir im HTML-Code direkt unter dem <body> das <header>-Tag. Für großen und fetten Text nutzen wir darin das <h1>-Tag.

HTML-Struktur im VS Code:

<header>
    <h1>Azure Cloud Network Control Center</h1>
</header>

CSS-Styling im Style-Tag:

header {
    background-color: var(--azure-blue);
    color: white;
    padding: 20px;
    text-align: center; /* Zentriert den Text */
}

💻 Das Navigationsmenü (<nav>) erstellen

Das <nav>-Tag wird verwendet, um den Hauptmenübereich der Website zu definieren. Die Links zu den Abschnitten werden mit dem Anker-Tag <a> erstellt.

HTML-Struktur im VS Code:

<nav>
    <a href="#overview">Übersicht</a>
    <a href="#subnets">Subnetze</a>
</nav>

CSS-Styling (Unterstreichung entfernen):

nav a {
    color: #f3f2f1;
    text-decoration: none; /* Entfernt die Unterstreichung */
    font-weight: bold;
    margin: 0 15px;        /* Abstand zwischen den Links */
}

💻 VS Code Anleitung: Die Willkommen-Box (Rechteck-Form)

Tags wie <div> sind Block-Elemente. Sie verhalten sich standardmäßig wie ein Rechteck und nutzen die volle Breite. Mit CSS machen wir dieses Rechteck sichtbar und stylen es modern.

HTML-Struktur im VS Code:

<div class="welcome-box">
    <h2>Willkommen...</h2>
</div>

CSS-Styling für das Rechteck:

.welcome-box {
    background: var(--card-bg);
    padding: 20px;              /* Innerer Abstand */
    border-left: 5px solid var(--azure-blue); /* Blauer Balken links */
    border-radius: 4px;         /* Abgerundete Ecken */
}

💻 VS Code Anleitung: Spalten erstellen mit CSS-Grid

Ein <div>-Tag ist eine neutrale Box. Damit mehrere Divs (wie unsere Status-Karten) automatisch als Spalten nebeneinander stehen, nutzen wir das moderne CSS-Grid im übergeordneten Container.

HTML-Struktur im VS Code:

<div class="status-grid">
    <div class="card">Spalte 1</div>
    <div class="card">Spalte 2</div>
    <div class="card">Spalte 3</div>
</div>

CSS-Steuerung für automatische Spalten:

.status-grid {
    display: grid; /* Aktiviert das Raster-Layout */
    /* Erstellt Spalten automatisch je nach Bildschirmgröße: */
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;     /* Abstand zwischen den Spalten */
}

💻 Status-Karten (Cards) mit Divs erstellen

Ein Div ist ein Container oder eine Box, mit der wir die Seite in Abschnitte unterteilen. Durch CSS geben wir diesen Boxen das Aussehen einer modernen Karte.

HTML-Struktur im VS Code:

<div class="card">
    <h3>Azure Firewall</h3>
    <span class="status-badge">Gesichert</span>
</div>

CSS-Code im Style-Tag:

.card {
    background: var(--card-bg);
    padding: 20px;
    border-radius: 4px;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

💻 Strukturierte Daten mit Tabellen (<table>)

Um Netzwerkdaten wie IP-Bereiche sauber in Zeilen und Spalten darzustellen, verwenden wir das <table>-Tag. Eine HTML-Tabelle wird immer zeilenweise aufgebaut.

Die wichtigsten Tabellen-Tags:

  • <tr> (Table Row): Erstellt eine horizontale Zeile im Raster.
  • <th> (Table Header): Erstellt die Kopfzeile (die Spaltenüberschriften). Der Text wird automatisch fett.
  • <td> (Table Data): Erstellt eine einzelne Zelle in einer normalen Zeile für die Daten.

HTML-Struktur im VS Code:

<table>
    <tr>
        <th>Subnetz-Name</th>
        <th>Adressbereich</th>
    </tr>
    <tr>
        <td>Subnet-Application</td>
        <td>10.0.1.0/24</td>
    </tr>
</table>
